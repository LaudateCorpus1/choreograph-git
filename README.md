Les modules (nommés par des nom de fichier) doivent avoir des noms courts tout en minuscules. Les traits de soulignement peuvent être utilisés dans le om du module s'il améliore la lisibilité.

Les paquets (nommés par des nom de repertoire)  doivent également avoir des noms cours tout en minuscule, mais sans tiré.

Tout fichier python (.py) est un module et un groupe de module dans un repertoire est un paquet.
Dans chaque repertoire de paquet, un fichier __init__.py doit etre présent (meme vide)
